### 2.4 配额管理

```go
// 配额检查逻辑
type QuotaChecker struct {
    maxSamplesPerSecond int
    maxSeries          int
    maxStorage         string
}

func (q *QuotaChecker) Check(ctx context.Context, tenantID string, samples int) error {
    // 1. 检查 QPS 配额
    currentQPS, _ := q.redis.GetQPS(tenantID)
    if currentQPS + samples > q.maxSamplesPerSecond {
        return ErrQuotaExceeded // 返回 429
    }
    
    // 2. 检查 Series 配额
    currentSeries, _ := vmClient.GetSeriesCount(ctx, tenantID)
    if currentSeries > q.maxSeries {
        return ErrSeriesQuotaExceeded
    }
    
    // 3. 检查存储配额
    storageUsed, _ := q.k8sClient.GetPVCSize(tenantID)
    storageLimit, _ := parseResource(q.maxStorage)
    if storageUsed >= storageLimit {
        return ErrStorageQuotaExceeded
    }
    
    return nil
}
```

## 6. 数据隔离验证

### 6.1 租户隔离检查

```go
// 验证租户数据隔离
func VerifyTenantIsolation(tenantID string) error {
    // 1. 验证 Namespace 隔离
    ns, err := k8sClient.GetNamespace(tenantID)
    if err != nil {
        return fmt.Errorf("Namespace 不存在: %v", err)
    }
    
    // 2. 验证 NetworkPolicy
    np, err := k8sClient.GetNetworkPolicy(tenantID)
    if np == nil {
        return fmt.Errorf("NetworkPolicy 未配置")
    }
    
    // 3. 验证 VM tenant_id
    series, err := vmClient.Query(ctx, "vm_labels{tenant_id!='"+tenantID+"'}")
    if len(series) > 0 {
        return fmt.Errorf("数据隔离验证失败: 存在跨租户数据")
    }
    
    // 4. 验证 Grafana 组织隔离
    orgUsers, _ := grafanaClient.ListOrgUsers(tenantID)
    // 确认只有租户内用户
    
    return nil
}
```

### 6.2 数据流安全检查

| 检查项 | 检查方式 | 失败处理 |
|--------|---------|---------|
| Tenant ID 伪造 | 验证 JWT/API Key | 拒绝请求 |
| 跨租户访问 | NetworkPolicy | 隔离流量 |
| 越权查询 | Tenant ID 过滤 | 返回空 |
| 数据泄露 | 审计日志 | 告警通知 |
